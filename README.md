W4 Inflow Configuration Inspector
=================================

This is a small web tool allowing to inspect and modify the configuration (key-value pairs) of W4 Inflow (v9.4+).

Beware: hacking a configuration with this tool may result in an incoherent state.


Try it live
-----------

As it's pure HTML/JS you can [try it live](https://rawgit.com/w4software/w4-inflow-tool-config-inspector/live-demo/index.html)
and enter the credentials to an installation of W4 BPMN+ Engine you have.


Download
--------

You can either download [a zip archive of the source code](https://github.com/w4software/w4-inflow-tool-config-inspector/archive/master.zip) or clone it using Git.


Build
-----

Run the following

    npm run build

This will install the necessary dependencies and wire them in the index file. Then deploy it on an HTTP server to run it.
